# iso_3166
> Lookup information with ISO 3166-1 alpha-2, ISO 3166-1 alpha-3 and ISO 3166-1 numeric

## Installation
Add `iso` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:iso_3166, "~> 1.0.0"}]
end
```
